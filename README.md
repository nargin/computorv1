# Computor

A quadratic equation solver written in Zig.

## Building

```bash
zig build
```

## Testing

```bash
zig build test
```

## Running

```bash
./computor "equation"
```

Example:
```bash
./computor "5 * X^2 + 3 * X + 2 = 1 + X"
```

## Requirements

- Zig compiler
